# Key bindings
```bash
Ctrl-o # go to previous (old) location
Ctrl-i # go to next location

# NERDTree
r # refresh

# windows
Ctrl-w <arrow up/down/left/right> # move cursor to another window
Ctrl-w = # resize all windows equally

# reindent
gg=G # reindent from top to bottom

# tabs
gt           # go to next tab
gT           # go to previous tab
{i}gt        # go to tab in position i
```

# Commands
```bash
:e filename      # edit another file

# search
:Ack             # call 'ag' to search for text

# windows
:split filename  # split window horizontally and load another file
:vsplit filename # split window vertically and load another file
```
